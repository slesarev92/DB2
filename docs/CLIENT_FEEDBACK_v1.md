## ЛОГИКА (расчёты, бизнес-правила)

### LOGIC-01 — Copacking в расчётах (Slides 11, 36) ⭐ ВАЖНО

**Описание:** Сейчас copacking = 0 в pipeline. Заказчик:
- Если SKU производится на копакере → Production cost отсутствует, вместо него — тариф копакера (₽/ед или ₽/л)
- Если SKU на своём производстве → Production cost = % от цены отгрузки
- **Косты копакера и своего производства одновременно не существуют**
**Влияние:** Критично для COGS и всех downstream KPI.
**Где:** `backend/app/engine/steps/s03_cogs.py`, `PipelineInput`, model `ProjectSKU`.
**Что нужно:**
1. Добавить поле `production_mode: "own" | "copacking"` на ProjectSKU
2. Если `copacking` → использовать copacking_rate (₽/ед), production_cost_rate = 0
3. Если `own` → использовать production_cost_rate (%), copacking = 0
4. UI: radio button в SKU параметрах, условное отображение полей
